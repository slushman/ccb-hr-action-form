# HR Action Form
React app that allows leaders within CCB to post an HR action form.



## Tech learned/practiced in this project:
* React
* Redux
* Firebase
* Material UI
* Formik
* Yup
* Jest & Enzyme
* day.js (removed moment.js)
* Styled Components
* memoize
* axios (removed)
* React Context API (removed)
