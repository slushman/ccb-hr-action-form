# HR Action Form
React app that allows leaders within CCB to post an HR action form.



## Tech learned/practiced in this project:
* React
* React Context API
* Firebase
* Material UI
* Formik
* Yup
* Jest & Enzyme
* Axios
* Moment.js
* Styled Components
